# Servo Motor With Keypad And LCD
Servo Arm Mover With Keypad And I2C LCD 16x2.

## Prerequisites
1. Arduino Uno R3
2. Arduino IDE
3. I2C LCD
4. Keypad (4*4)
5. Jumper Wires
6. Breadboard / PCB
